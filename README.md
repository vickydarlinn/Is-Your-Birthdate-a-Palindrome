# Is-Your-Birthdate-a-Palindrome
This web app checks your birthdate in 4 formats DD-MM-YYYY , MM-DD-YYYY , YYYY-MM-DD , DD-MM-YY , MM-DD-YY , YY-MM-DD. e.g. if your birthdate is 10th september 2020, then app will check for '10092020', '09102020', '20200910', '091020', '100920', '200910'
### [Live preview](https://birthday-palindrome-checker-neog.netlify.app/)
## Built With
* HTML
* CSS
* JS
